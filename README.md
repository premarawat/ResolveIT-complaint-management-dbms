resolveIT is a Database Management System (DBMS) project designed to
provide a fair, transparent, and structured complaint resolution system
for smart cities.

## Problem Statement
City complaint systems often lack transparency, accountability, and
proper tracking mechanisms. resolveIT aims to solve this by maintaining
structured relational data for citizens, complaints, departments, and
administrators.

## Modules
- Citizens
- Complaints
- Departments
- Categories
- Admins
- Audit Logs

## Features
- Structured relational database design
- Complaint tracking and status management
- Department-wise complaint allocation
- Audit logs for transparency

## Technologies Used
- MySQL
- CSV datasets
- ER Diagram
